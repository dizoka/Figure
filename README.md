# ЛНТУ калькулятор конкурсного бала

Render: Web Service

Build Command: `npm install`
Start Command: `npm start`

Важливо: після заміни файлів на GitHub натиснути Render → Manual Deploy → Clear build cache & deploy.

Що виправлено:
- пропозиції не змішуються за однією спеціальністю;
- J8 має окремі рядки для різних освітніх програм і форм;
- якщо для пропозиції немає підтвердженого мінімального бюджету, сайт не підставляє чужий бал;
- якщо Освіта.UA блокує live-запит з Render, сайт бере великий embedded snapshot, а не 4/19 рядків.
