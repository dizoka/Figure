# ЛНТУ — калькулятор конкурсного бала

Чистий Web Service для Render.

## Render
- New + → Web Service
- Build Command: `npm install`
- Start Command: `npm start`

## Що важливо
- Дані пропозицій вбудовані в сайт як snapshot, щоб Render не падав через 403 від osvita.ua.
- Кожна конкурсна пропозиція — окремим рядком: спеціальність, освітня програма, форма, рівень, бюджетні місця, мінімальний бюджетний бал.
- Розрахунок НМТ зроблено за логікою Освіта.UA: K4max у знаменнику, окремо 1–2 та 3–10 пріоритети.
